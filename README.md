# Ні! котину мотиватор

Невеликий мотиваційний WinForms-застосунок на C#/.NET 8.

## Що вміє

- зберігає дату й час відмови від куріння;
- зберігає кількість пачок на день;
- зберігає ціну пачки в гривнях;
- автозберігає зміни в конфіг після редагування;
- рахує й показує заощаджену суму у форматі `грн/коп`;
- великим шрифтом показує час без куріння;
- показує зверху бігучий рядок із локального довідника мотиваційних фраз;
- показує 3 товари з локального каталогу, які можна купити на поточну суму;
- показує нижній таймлайн змін в організмі після відмови від куріння: що вже пройдено, що актуально зараз і що далі;
- оновлює добірку товарів вручну і кожні 30 хвилин;
- оновлює суму в реальному часі після запуску.

## Запуск

```powershell
dotnet build .\NitKotin\NitKotin.csproj
dotnet run --project .\NitKotin\NitKotin.csproj
```

## Конфігурація

Файл конфігурації створюється поруч із застосунком:

`nitkotin.config.json`

Якщо файлу немає або він пошкоджений, застосунок запускається зі значеннями за замовчуванням.

## Локальний каталог товарів

Застосунок містить локальний каталог прикладів товарів у файлі:

`Data\product-catalog.ua.json`

Це не live-ціни магазину, а мотиваційні приклади в діапазоні від 100 до 20000 грн.

## Довідник мотиваційних фраз

Локальний довідник фраз лежить у файлі:

`Data\motivational-phrases.ua.json`

Застосунок показує ці фрази у верхньому бігучому рядку. Щоб додати нову фразу, достатньо додати новий об'єкт у JSON у форматі:

```json
{ "Text": "Нова мотиваційна фраза", "Category": "Здоров'я" }
```

Якщо файл відсутній або пошкоджений, застосунок використовує вбудований запасний список фраз.

## Таймлайн відновлення

У нижньому блоці застосунок показує консервативний таймлайн типових змін після відмови від куріння. Це не медичний прогноз для конкретної людини, а зрозумілі орієнтири прогресу: уже пройдені етапи, поточний етап і найближчі наступні зміни.
